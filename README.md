# ManishMart

Simple static e-commerce demo site.

Files:
- `index.html` — main page
- `style.css` — styles
- `script.js` — site scripts
- `images/` — assets

Getting started:
1. Open `index.html` in a browser to preview locally.

To push to GitHub (recommended):

Using GitHub web UI:
- Create a new repository on GitHub.
- Follow the instructions to push an existing repository from the command line.

Using GitHub CLI (`gh`):
```bash
gh repo create manishmart --public --source . --push
```
